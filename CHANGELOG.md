## [v0.3.0] - 2026-01-12

### Highlights

- Complete transformation from pre-push validation tool to release automation platform with workflow engine, Claude Code integration, and multi-platform plugin support

- Release workflow engine with 9-step process: validate → check → changelog → commit → push → CI wait → tag ([`960f1c8`](https://github.com/grokify/release-agent/commit/960f1c8))
- Comprehensive validation command with 4 areas: QA, Documentation, Release, Security ([`554e12a`](https://github.com/grokify/release-agent/commit/554e12a))

- Report package (`pkg/report`) with structured data types and text/template renderer ([`0ce034f`](https://github.com/grokify/release-agent/commit/0ce034f))
