## [v0.5.0] - 2026-01-17

### Highlights

- Reports display teams in DAG order (PM first, parallel teams alphabetically, Release last)

- Typed inputs/outputs for workflow steps in team spec enabling data flow between agents ([`3e589b7`](https://github.com/agentplexus/release-agent-team/commit/3e589b7))
- Runtime configuration in deployment spec for distributed mode (timeouts, retries, observability) ([`3e589b7`](https://github.com/agentplexus/release-agent-team/commit/3e589b7))
- New deployment targets: `gemini-cli`, `adk-server`, `k8s-production` ([`3e589b7`](https://github.com/agentplexus/release-agent-team/commit/3e589b7))
